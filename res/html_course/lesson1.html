1.Basic HTML Elements

Headings

HTML headings are defined with <h1> to <h6> tags.

Big heading(h1)

H2 heading

...
Little heading(h6)

Paragraphs

Paragraphs are defined with the <p> tag.

Syntax:

<p>This is a paragraph</p>

Example:

This is a paragraph.

This is another paragraph.

Links

Syntax:

<a href="https://www.google.com">Click here to go to Google</a>

Example:

Click here to go to Google

Syntax:

Line break (new line)

The element used for line breaks is <br>.

Syntax:

Hey!<br> I am learning HTML

Example:

Hey!
I am learning HTML
Elements like <br> and <img> are empty elements (they don't have content) and they have no end tag.
